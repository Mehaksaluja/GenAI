# Mini ChatGPT (LangChain + Hugging Face)

A lightweight local chatbot built using LangChain and Hugging Face FLAN models.

## Features
- Free & offline
- Local model inference
- LangChain runnable chains
- Customizable prompt templates

## Setup
1. Clone the repo
2. Create a virtual environment
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
4. Run the app
    python app.py
